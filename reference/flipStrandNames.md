# Reverse order and flip strand

Takes a character vector of "strand names", reverses the order of its
elements and flips the strand.

## Usage

``` r
flipStrandNames(x)
```

## Arguments

- x:

  A vector of strand names.

## Value

Returns the same class as its input.
