# Plot Alignment of Single Reference Scaffold

The function plots the pairwise alignments from one of the reference
scaffolds

## Usage

``` r
scaf_align_plot(gr_ob, scaf)
```

## Arguments

- gr_ob:

  GRanges object containing pairwise alignment

- scaf:

  Character string of the name of the scaffold you wish to plot

## Value

A plot of the query scaffold alignments to the chosen reference scaffold
