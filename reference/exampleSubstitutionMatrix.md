# Example Substitution Matrices

The probability matrix is loaded from the file `extdata/example.train`
distributed in this package.

## Usage

``` r
exampleSubstitutionMatrix
```

## Format

An object of class `matrix` (inherits from `array`) with 6 rows and 6
columns.
