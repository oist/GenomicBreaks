# Attempt to load a BSgenome

Attempt to load a BSgenome

## Usage

``` r
getRefGenome(reference.genome)
```

## Arguments

- reference.genome:

  name of genome package (and also genome data object)

## Value

A BSgenome object

## Details

Internal function that retrieves a BSgenome object or throws an error if
not available. Adapted from the CAGEr package.
